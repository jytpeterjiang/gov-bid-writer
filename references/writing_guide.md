# 各章节详细写作指南和范例

## 一、项目理解（1.1节）写作范例

### 写作框架

1. 第一段：项目来源和基本概况
2. 第二段：核心需求拆解
3. 第三段：评审分析和我方策略

### 范例

```
本项目为"[项目名称]"，由[采购单位]组织自行采购，预算金额[XX]万元，
服务周期为[周期]。

项目核心需求分为[N]大板块：一是[需求一概述]；二是[需求二概述]；
三是[需求三概述]。[如有特殊要求在此强调，如"需支持信创环境"]

项目评审采用综合评分法，采购方案整体评价占[XX]分（权重最高），
团队综合能力[XX]分，公司实力[XX]分，同类业绩[XX]分，公司诚信[XX]分，
价格[XX]分。这要求我方在采购方案上做到全面、具体、科学、可操作，
逐条响应"[相关章节号]"所有项，方能获得技术方案高分。
```

## 二、服务方案响应写作范例

### 售后响应（范例）

```
【招标要求】售后热线7×24小时受理。

【我方响应】我方设立7×24小时售后热线（【热线电话：待补充】），
全年365天、每天24小时受理用户问题。热线受理实行首问负责制，
建立统一的工单管理系统，来电必录、有问必答。同时开通微信公众号/
小程序在线报修通道和专属企业微信服务群，确保用户可通过多种渠道
快速触达售后团队。

故障分级响应体系：

| 故障级别 | 定义 | 响应时限 | 解决时限 | 报告要求 |
|---------|------|---------|---------|---------|
| 一级（严重） | 系统不可用、核心功能瘫痪 | 30分钟内 | 4小时内 | 故障报告+根因分析 |
| 二级（紧急） | 部分功能不可用 | 1小时内 | 8小时内 | 故障报告 |
| 三级（一般） | 个别功能异常 | 4小时内 | 24小时内 | 纳入月报 |
| 四级（咨询） | 操作咨询 | 8小时内 | 48小时内 | 纳入月报 |
```

### 服务升级/补丁管理（范例）

```
【招标要求】标准软件升级、后台补丁安装。每年不少于6次。

【我方响应】
（1）标准软件升级服务。当腾讯发布政务微信新版本后，我方在3个
工作日内完成版本评估报告（包含新功能清单、兼容性影响分析、
已知问题列表）。升级安排在非工作时段执行，采用分批次灰度发布
策略，先覆盖10%试点用户观察48小时，确认无异常后逐步放量。
每次升级全程录屏留档，升级结束后24小时内提交升级报告。

（2）后台补丁安装服务。按招标要求每年不少于6次补丁安装。
我方制订季度补丁计划（每季度至少1-2次），安全补丁发布后
48小时内启动评估；常规功能补丁安排在月度维护窗口统一安装。
补丁安装前在测试环境完成72小时验证，安装完成后进行全量
功能回归测试。
```

### 工单驱动（写作模式）

描述任何运维/服务流程时，可以引入"工单驱动"概念来体现代谢化：
```
所有服务请求通过统一工单系统管理，每个工单有唯一编号可追溯，
工单类型涵盖：故障报修、功能咨询、需求变更、账号管理、
配置调整、巡检任务等。工单处理完成后须甲方管理员确认方可关闭。
```

### 备份机制（差异化亮点）

```
我方为驻场人员配备1名影子工程师（不驻场但全程同步了解项目情况），
在驻场人员因病/事假无法到岗时，影子工程师在1个工作日内接管
全部工作，确保服务不中断。
```

## 三、费用明细写作

- 使用5列表格（序号、费用项目、服务内容、报价、备注）
- 所有费用项累加 = 总报价
- 总报价 ≤ 预算金额
- 备注列标注对应的方案章节号（如"详见1.3节"）

## 四、承诺函写作

### 符合资质要求的承诺函

必须逐条包含：
1. 政府采购法第二十二条6项要求（逐条列出）
2. 无被禁参与政府采购声明
3. 无失信记录声明
4. 无严重违法失信声明（按当地规定）
5. 关联关系合规声明
6. 真实性承诺

落款格式：
```
承诺人（公章）：【投标人全称】
法定代表人（签字）：_______________
日期：【    年    月    日】
```

### 诚信承诺函

在"八、采购人要求的其他材料"中，额外提供诚信承诺函（对应评分中"公司诚信"5分）。

内容包含：
1. 提交资料真实有效
2. 无串通投标行为
3. 投标有效期内不撤标
4. 中标后按要求履行
5. 如不实愿承担法律责任

## 五、法定代表人授权委托书写作

标准法律文本要素：
- 委托人（法定代表人）姓名、单位
- 被授权人姓名、身份证号、职务
- 授权事项：投标、开标、评标、合同签署、合同执行
- 无转委托权
- 有效期：签署之日至合同履行完毕
- 附：双方身份证复印件（加盖公章）

## 六、避免的写作问题

### ❌ 空泛表述（避免）
- "我们将提供优质的服务"
- "我们有丰富的经验"
- "我们保证系统稳定运行"
- "我们会及时响应"

### ✅ 具体表述（追求）
- "我方设立7×24小时热线，30分钟内响应一级故障"
- "我公司已完成15个同类政务微信项目，合同总额超1200万元"
- "每日自动巡检3次（6:00、12:00、18:00），每周人工全面巡检1次"
- "一级故障30分钟响应、4小时解决；二级故障1小时响应、8小时解决"
