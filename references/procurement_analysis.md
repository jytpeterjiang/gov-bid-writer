# 招标文件解读方法论

## 一、招标文件的核心组成

一份标准的政府采购自行采购公告通常包含以下部分：

### 1. 基本信息区
- 索引号、发布机构、成文日期、发布日期
- 这些信息帮助确认招标的正式性和时效性

### 2. 项目基本情况
- **项目名称**：精确的项目名称（这是标书封面的必填项）
- **预算金额**：这是报价的上限，超过预算会废标
- **采购需求**：这是核心——所有技术方案必须逐条响应的内容
- **采购方式**：公开征集/公开招标/竞争性谈判等
- **服务周期**：合同履行的时间范围
- **是否接受联合体**：大多数政府项目不接受联合体

### 3. 供应商资格要求
- 逐条列出，通常包括：
  - 《政府采购法》第二十二条
  - 独立法人资格
  - 无被禁参与政府采购
  - 无失信记录
  - 关联关系限制
- 这些需要在承诺函中逐条声明

### 4. 其他服务要求（技术需求）
- 这是采购方案要逐条响应的核心内容
- 按模块组织，每个模块有多个子项
- 注意区分"必须满足"和"建议满足"

### 5. 提交文件清单
- 采购方案
- 资质证明材料
- 法人身份证
- 授权委托书
- 承诺函
- 服务能力证明
- 基本情况表
- 其他材料

### 6. 评定方式
- **综合评分法** vs 最低价法
- 综合评分法需要关注每个评分因素的权重

### 7. 提交方式和截止时间
- 电子邮箱 / 纸质文档
- 截止时间（逾期无效）

### 8. 评审评分规则
- 评分因素
- 评分规则说明
- 权重

## 二、评分规则深度解析

### 典型评分结构

| 评分因素 | 典型权重 | 得分策略 |
|---------|---------|---------|
| 价格 | 30 | 低价优先法，报接近预算但略低于竞争者的价格 |
| 采购方案整体评价 | 35-40 | 逐条响应，全面具体，科学合理，操作性强 |
| 项目团队综合能力 | 10-11 | 准备充分的证书和社保材料 |
| 公司实力 | 8-10 | 准备相关认证证书 |
| 同类项目业绩 | 5-6 | 准备近3年的同类合同 |
| 公司诚信 | 5 | 提供承诺函即可 |

### 评分等级标准

技术方案通常分四档：
- **优（满分）**：技术方案全面具体、科学合理、操作性强
- **良（80%分）**：技术方案较全面，但有一项不够具体，方案基本合理、有可操作性
- **中（60%分）**：有具体内容，但一项以上不够具体，科学合理一般、操作性一般
- **差（0分）**：未提供或不全面、合理性有明显缺陷、操作性不强

这意味着"具体"和"全面"是拉开差距的关键！

### 客观分 vs 主观分

- **客观分**（有就有，没有就没有）：团队证书、公司资质、同类业绩
- **主观分**（评委打分）：采购方案整体评价

客观分要通过充分准备材料来确保满分。
主观分要通过详细、具体、专业的方案来争取满分。

## 三、需求逐条分析法

对于招标文件中的每一条需求，使用以下框架分析：

1. **需求类型**：必须满足 / 加分项
2. **对应标书章节**：该需求在标书哪一节响应
3. **响应策略**：
   - 逐字逐句对应，使用【招标要求】引用原文
   - 给出比我方承诺更具体的实施方案
   - 使用量化指标（时间、数量、频率、百分比）
4. **证明材料**：需要哪些证书/合同/截图支撑

## 四、招标文件中的关键词识别

- "逐条响应" / "逐项响应"：意味着必须用显式的对应结构
- "具体" / "详细"：意味着需要给出操作层面的实施方案
- "科学合理"：意味着方案要有方法论支撑
- "操作性强"：意味着方案要落地、可执行
- "原件备查"：标书中放复印件，但原件要准备好
- "需盖公章"：不能忘！
